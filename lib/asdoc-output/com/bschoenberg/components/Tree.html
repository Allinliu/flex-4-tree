Packagecom.bschoenberg.components
Classpublic class Tree
InheritanceTree spark.components.List

Flex 4 Spark Tree component. It extends List. Its item renderers must implement ITreeLayoutElement. It does not use a dataProvider, but rather a dataSource which much implement the ITreeDataSource interface. All items renderered by the Tree must be of type ITreeItem

Default MXML PropertydataProvider
