Interfacepublic interface ITreeDataSource extends flash.events.IEventDispatcher
Implementors TreeDataSource

This interface describes a Tree data structure which is used as the source of data for the Tree component It acts on items which implement the ITreeItem interface. Any changes to this data structure will change the visual output of the Tree.

addItem()method
public function addItem(item:ITreeItem, parent:ITreeItem = null, dispatchTreeEvent:Boolean = true):void

This method adds an Item at any point in the data structure. The item will be added as the last child for the specified parent.

Parameters

item:ITreeItem — The item to add to the data source
 
parent:ITreeItem (default = null) — The parent item of the item being added. If null this item will be top level
 
dispatchTreeEvent:Boolean (default = true) — Whether or not the datastructure should dispatch a TreeEvent causing an animation or state change
