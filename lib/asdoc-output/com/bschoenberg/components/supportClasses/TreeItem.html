Classpublic dynamic class TreeItem
InheritanceTreeItem flash.events.EventDispatcher
Implements ITreeItem

This class is the main implementor of ITreeItem. It is the item that will be passed to the TreeItemRenderer It is dynamic so that you may add any arbitrary data to it to be displayed by the tree.
